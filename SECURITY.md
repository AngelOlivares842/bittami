# Security Policy

## Supported Versions

Actualmente, solo se brinda soporte de seguridad a la última versión estable del proyecto **BittamiHub**.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Como estudiante de ingeniería, valoro profundamente la integridad del sistema. Si descubres una vulnerabilidad de seguridad en este proyecto, por favor sigue estos pasos:

1. **No abras un "Issue" público:** Esto podría exponer el sistema a ataques antes de que pueda ser reparado.
2. **Envía un reporte detallado:** Por favor, contacta directamente a través de:
   - **Email:** angel.olivares.rosas2@gmail.com
   - **Asunto:** SECURITY_VULNERABILITY_REPORT - [Tu Nombre/Alias]

### Información requerida en el reporte:
- Descripción del posible impacto.
- Pasos para reproducir la vulnerabilidad (PoC).
- Versión del software afectada.

### Nuestro Compromiso
Me comprometo a revisar el reporte en un plazo de **48-72 horas** y a trabajar en un parche de seguridad antes de realizar cualquier divulgación pública. Se dará el crédito correspondiente al investigador en las notas del parche.
